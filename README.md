# Multiplayer-Terminal-BattleShip
Multiplayer Battleship game that runs on the terminal

`IMPORTANT NOTE YOU MUST USE "Radmin VPN" FOR THE "Radmin Multiplayer" TO WORK,
And for the local multiplayer to work you need to be both on the same network
`
#### To Quit the game press `Escape`
